# Installation

Install `@exassess/drizzle-query-resource` alongside `drizzle-orm`.

## Install the package

::code-group
```bash [bun]
bun add @exassess/drizzle-query-resource drizzle-orm
```

```bash [npm]
npm install @exassess/drizzle-query-resource drizzle-orm
```

```bash [pnpm]
pnpm add @exassess/drizzle-query-resource drizzle-orm
```

```bash [yarn]
yarn add @exassess/drizzle-query-resource drizzle-orm
```
::

`drizzle-orm` is a peer dependency.

## Prepare the three required inputs

Before you define resources, gather:

- your Drizzle database instance as `db`
- your schema map as `schema`
- your relations map as `relations`

These three objects let the engine infer valid root tables, relation paths, and default row types.

## Keep your schema and relations in sync

The engine assumes the schema map and relations object describe the same graph.

If they drift apart, you can end up with invalid assumptions about:

- nested field paths
- row hydration shape
- relation traversal

## Run the package quality checks before release

This package uses Bun-oriented scripts for local verification.

::code-group
```bash [bun]
bun run test
bun run typecheck
bun run build
```

```bash [npm]
npm run test
npm run typecheck
npm run build
```
::
