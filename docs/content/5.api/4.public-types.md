# Public Types

The package exports a broad set of public types from `index.ts`.

## Reach for the most common integration types first

The types most users touch directly are:

- `QueryEngine`
- `QueryEngineConfig`
- `QueryRequest`
- `QueryResponse`
- `QueryIdsResponse`
- `QueryFacetsResponse`
- `QueryFacetRequest`
- `QueryFieldPath`
- `QueryFilterBuilder`
- `QueryFilterCondition`
- `QueryFilterGroup`
- `QueryFilterNode`
- `QueryFilterInput`
- `QueryResource`
- `QueryResourceUtils`
- `ResourceQueryConfig`
- `ResourceQueryStrategyConfig`

## Use row and relation helpers when you need advanced typing

For resource and strategy typing, these helpers are especially useful:

- `QueryWith`
- `QueryRowShape`
- `QueryResultRowShape`
- `QueryRootKey`
- `DefineResourceOptions`
- `DefineQueryResourceOptions`

## Keep your imports at the package root

```ts [types.ts]
import type {
  QueryRequest,
  QueryResponse,
  QueryResource,
  ResourceQueryStrategyConfig,
} from '@exassess/drizzle-query-resource'
```
