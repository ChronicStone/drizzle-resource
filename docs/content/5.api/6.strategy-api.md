# Strategy API

The strategy surface has four hooks: `query`, `ids`, `rows`, and `facets`.

## Implement every available hook in one resource

```ts [resource.ts]
const employees = engine.defineResource('employees', {
  relations: {
    department: {
      with: {
        company: true,
      },
    },
    employeeSkills: {
      with: {
        skill: true,
      },
    },
  },
  strategy: {
    query: async (args) => ({ rows: [], rowCount: 0, facets: [] }),
    ids: async (args) => ({ ids: [], rowCount: 0 }),
    rows: async (args) => [],
    facets: async (args) => ({ facets: [] }),
  },
})
```

## Shared args for `query` and `ids`

```ts [types.ts]
{
  request: QueryRequest
  context?: TContext
  resource: QueryResource<TDb, TSchema, TRelations, TRoot, TWith, TContext, TRow>
  utils: QueryResourceUtils<TRow>
}
```

## Extra args for `rows` and `facets`

`rows` also receives:

```ts [types.ts]
{
  ids: Array<TRow extends { id: infer TId } ? TId : unknown>
}
```

`facets` also receives:

```ts [types.ts]
{
  facets: QueryFacetRequest<TFacetKey>[]
}
```

## Match the exact return shape

### `query`

```ts [types.ts]
Promise<QueryResponse<TRow>>
```

### `ids`

```ts [types.ts]
Promise<QueryIdsResponse<TRow extends { id: infer TId } ? TId : unknown>>
```

### `rows`

```ts [types.ts]
Promise<TRow[]>
```

### `facets`

```ts [types.ts]
Promise<QueryFacetsResponse<TFacetKey>>
```

## Use the built-in helpers from `utils`

- `utils.buildWhereClause(request)`
- `utils.compileOrderBy(request.sorting)`
- `utils.executeRowsQuery({ ids, request })`
- `utils.resolveFacets({ request, facets })`
