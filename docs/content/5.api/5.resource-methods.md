# Resource Methods

Every resource exposes four public methods.

## Use `query(...)` for the full pipeline

```ts [query.ts]
const response = await employees.query({
  context: { orgId: 'fr' },
  request: {
    pagination: { pageIndex: 1, pageSize: 25 },
    sorting: [{ key: 'fullName', dir: 'asc' }],
    search: { value: 'ada', fields: [] },
    filters: {
      type: 'group',
      combinator: 'and',
      children: [],
    },
    context: {},
    facets: [
      {
        key: 'department.name',
        mode: 'exclude-self',
        limit: 10,
      },
    ],
  },
})
```

Returns:

```ts [types.ts]
{
  rows: TRow[]
  rowCount: number
  facets?: QueryFacetResult[]
}
```

## Use `queryIds(...)` for ids and total count only

```ts [query-ids.ts]
const idsResult = await employees.queryIds({
  context: { orgId: 'fr' },
  request,
})
```

Returns:

```ts [types.ts]
{
  ids: Array<TRow extends { id: infer TId } ? TId : unknown>
  rowCount: number
}
```

## Use `queryRows(...)` when ids were selected elsewhere

```ts [query-rows.ts]
const rows = await employees.queryRows({
  context: { orgId: 'fr' },
  request,
  ids: ['emp_2', 'emp_5', 'emp_9'],
})
```

Returns:

```ts [types.ts]
Promise<TRow[]>
```

## Use `queryFacets(...)` when facet resolution is independent

```ts [query-facets.ts]
const facets = await employees.queryFacets({
  context: { orgId: 'fr' },
  request,
  facets: [
    {
      key: 'department.company.name',
      mode: 'exclude-self',
      limit: 20,
    },
  ],
})
```

Returns:

```ts [types.ts]
{
  facets: QueryFacetResult[]
}
```
