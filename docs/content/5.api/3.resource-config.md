# Resource Config

`defineResource(root, options)` is the main surface for shaping a release-ready resource.

::note
The detailed configuration reference now lives under the dedicated [Configuration](/configuration/overview) section, with one page each for relations, scope, search, sorting, filters, facets, pagination/defaults, and strategies.
::

## Set the root table

The `root` argument is the schema and query key for the table, for example:

- `'employees'`
- `'departments'`
- `'companies'`
