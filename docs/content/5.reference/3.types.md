# Types

All types are exported from the package root:

```ts twoslash
import type {
  QueryRequest,
  QueryResponse,
  // ...
} from "drizzle-resource";
```

## Request types

### `QueryRequest`

The full shape of a query request. See [Query Contract → Request Shape](/query-contract/request-shape).

```ts twoslash
import type { QueryRequest } from "drizzle-resource";

type QueryRequestExample = QueryRequest;
```

## Engine types

### `QueryEngine<...>`

Returned by `createQueryEngine(...)`. The generic parameters are inferred from your database, schema, and relations.

### `QueryResource<...>`

The type of a defined resource, returned by `engine.defineResource(...)`. Exposes `query`, `queryIds`, `queryRows`, `queryFacets`.

### `QueryFieldPath<TSchema, TRelations, TRoot, TWith>`

A type-level field path — a union of all valid dotted field path strings for a given resource root and relation tree. This is what powers the compile-time validation of sort keys, filter keys, and facet keys.
