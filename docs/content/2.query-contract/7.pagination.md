# Pagination

Pagination is client input under `request.pagination`.

## Send a standard page request

```ts [query.ts]
pagination: {
  pageIndex: 1,
  pageSize: 25,
}
```

## Know the normalization rules

- `pageIndex` is one-based
- invalid `pageIndex` falls back to the configured default or `1`
- invalid `pageSize` falls back to the configured default or `25`

## See a request with invalid values

```ts [query.ts]
await employees.query({
  context: { orgId: 'fr' },
  request: {
    pagination: {
      pageIndex: 0,
      pageSize: -10,
    },
    sorting: [],
    search: { value: '', fields: [] },
    filters: {
      type: 'group',
      combinator: 'and',
      children: [],
    },
    context: {},
  },
})
```

## Configure default pagination separately

If you want to set fallback page size or page index defaults, go to [Configuration > Pagination and Defaults](/configuration/pagination-and-defaults).
