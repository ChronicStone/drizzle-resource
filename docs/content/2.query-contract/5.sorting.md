# Sorting

Sorting is client input under `request.sorting`.

## Sort by one field

```ts [query.ts]
sorting: [{ key: 'fullName', dir: 'asc' }]
```

## Sort by a nested relation field

```ts [query.ts]
sorting: [{ key: 'department.company.name', dir: 'asc' }]
```

## Use sorting inside a full request

```ts [query.ts]
await employees.query({
  context: { orgId: 'fr' },
  request: {
    pagination: { pageIndex: 1, pageSize: 25 },
    sorting: [{ key: 'department.company.name', dir: 'asc' }],
    search: { value: '', fields: [] },
    filters: {
      type: 'group',
      combinator: 'and',
      children: [],
    },
    context: {},
  },
})
```

## Configure defaults and disabled sort fields separately

If you want to choose default sorting or block unstable sort paths, go to [Configuration > Sorting](/configuration/sorting).
